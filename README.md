# Soccer Tournament Management System

A web application for organizing and managing soccer tournaments, including team registration, match scheduling, and standings tables.

## Technologies
- HTML5
- CSS3
- JavaScript

## Features
- Team registration and management
- Match scheduling
- Score tracking and results
- Live standings table
- Knockout bracket support

## Getting Started
Open `index.html` in your browser.
